About Stone Chat

Stone chat was written in a few hours on a nice sunny day in April. One big thing that needs to change about stone chat is that it is currently using regular polling. Ideally, it would use an open websockets port on the server and use websockets, but due to time contrains, it just pulls new chats from the api subdomain (if it works right).
